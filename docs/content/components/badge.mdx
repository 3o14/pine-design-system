# Badge

상태, 라벨, 카테고리 등을 시각적으로 표시하는 인라인 컴포넌트입니다.
텍스트 옆에 붙여 부가 정보를 전달하거나, 목록 아이템의 상태를 나타낼 때 사용합니다.

- **기본값**: `false`

## 사용 가이드

### 적절한 Variant 선택하기

- **solid**: 정보를 분명히 강조해야 할 때 사용합니다. 리스트 내 상태 표시, CTA 주변의 카운트 표시 등에 적합합니다.
- **outline / subtle / weak**: 텍스트 옆에 가볍게 붙이는 태그나 분류 라벨에 사용합니다. 화면 내 강조가 과해지지 않도록 조절하세요.

### Dot 사용하기

`showDot`은 실시간 상태를 나타내는 데 사용합니다. `success` ↔ 온라인/정상, `danger` ↔ 오류/오프라인, `warning` ↔ 주의 패턴을 권장합니다.

## 코드

```tsx
import { Badge } from "pine-design-system";

// 기본 사용
<Badge intent="primary">New</Badge>

// 상태 표시
<Badge intent="success" showDot>Active</Badge>
<Badge intent="danger" variant="outline">Error</Badge>

// 카테고리 태그
<Badge variant="subtle" intent="neutral">Design System</Badge>
```
