# Dropdown

여러 옵션 중 하나를 선택하는 드롭다운 선택 컴포넌트입니다.
[Base UI Select](https://base-ui.com/react/components/select)를 기반으로 키보드 탐색과 스크린리더 접근성을 기본으로 지원합니다.

선택지가 5개 이하이거나 모든 옵션을 동시에 보여줘야 할 때는 라디오 버튼을 고려하세요.
