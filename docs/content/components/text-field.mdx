# TextField

A form component for text input. Built on [Base UI Field](https://base-ui.com/react/components/field) and [Base UI Input](https://base-ui.com/react/components/input) to provide label, input area, and helper text in a single component. Use the `multiline` prop to render a `<textarea>` instead of `<input>`.

## Guidelines

### Showing error state

Use `status="error"` with `helperText` to explain what went wrong. A clear message helps users understand what to fix.

```tsx
<TextField
  label="Email"
  status="error"
  helperText="Please enter a valid email address."
  value={email}
  onChange={(e) => setEmail(e.target.value)}
/>
```
