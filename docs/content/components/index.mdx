# Components

Pine Design System이 제공하는 React 컴포넌트 목록입니다. 모든 컴포넌트는 `ThemeProvider` 하위에서 사용하며, `design` / `theme`에 따라 스타일이 적용됩니다.

## 목록

| 컴포넌트   | 설명                     |
| ---------- | ------------------------ |
| [Button](/components/button) | 버튼 (variant, intent, size) |
| [Badge](/components/badge)   | 뱃지·태그·상태 표시     |
| [Checkbox](/components/checkbox) | 체크박스               |
| [Dialog](/components/dialog) | 모달 다이얼로그         |
| [Dropdown](/components/dropdown) | 드롭다운 선택          |
| [Switch](/components/switch) | 토글 스위치            |
| [Tab](/components/tab)       | 탭 네비게이션          |
| [Text](/components/text)      | 타이포그래피           |
| [TextField](/components/text-field) | 텍스트 입력·textarea  |

각 컴포넌트는 `className`과 표준 HTML 속성을 지원하며, `forwardRef`로 ref를 전달할 수 있습니다.
