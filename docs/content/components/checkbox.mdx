# Checkbox

선택 입력용 체크박스 컴포넌트입니다. Base UI의 headless Checkbox를 기반으로 합니다.

## 예시

<ComponentPreview>
  <CheckboxDemo />
</ComponentPreview>

## Props

`Checkbox`는 Base UI Checkbox의 props와 호환되며, `label`을 주면 레이블이 함께 렌더링됩니다. `disabled`, `checked`, `defaultChecked` 등 표준 체크박스 속성을 지원합니다.

## 코드

```tsx
import { Checkbox } from "pine-design-system";

<Checkbox label="Accept terms" defaultChecked />
```
