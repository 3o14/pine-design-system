# Checkbox

사용자가 하나 이상의 옵션을 선택하거나 항목에 동의할 때 사용하는 선택 컨트롤입니다.
[Base UI Checkbox](https://base-ui.com/components/react-checkbox/)를 기반으로 접근성을 기본으로 지원합니다.

## 사용 가이드

### Controlled vs Uncontrolled

상태 관리가 필요 없는 단순한 경우 `defaultChecked`를 사용하는 것이 간단합니다.
폼 라이브러리나 외부 상태와 연동하거나, 체크 상태에 따라 다른 UI를 렌더링할 경우 `checked` + `onCheckedChange`를 함께 사용하세요.

### 접근성

`label` prop을 사용하거나, 직접 `<label>` 요소와 연결해 스크린리더가 Checkbox의 목적을 인식할 수 있도록 하세요.

## 코드

```tsx
import { Checkbox } from "pine-design-system";

// 기본 사용 (uncontrolled)
<Checkbox label="이용약관에 동의합니다" defaultChecked />

// Controlled
<Checkbox
  label="알림 수신"
  checked={checked}
  onCheckedChange={setChecked}
/>

// 비활성화
<Checkbox label="변경 불가" disabled defaultChecked />
```
