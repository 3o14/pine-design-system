# Text

일관된 타이포그래피를 적용하는 텍스트 렌더링 컴포넌트입니다.
`as` prop으로 시멘틱 HTML 요소를 지정할 수 있어 접근성과 SEO를 모두 고려한 마크업이 가능합니다.

## 사용 가이드

### 시멘틱 마크업 유지하기

`size`가 시각적 크기를 제어하고 `as`가 HTML 요소를 결정합니다. 두 가지가 독립적으로 동작하므로 SEO·접근성을 위한 적절한 시멘틱 구조를 유지하면서 디자인을 자유롭게 적용할 수 있습니다.

```tsx
// 시각적으로 large이지만 h2 태그로 렌더링
<Text as="h2" size="large" weight="semibold">섹션 제목</Text>

// 시각적으로 small이지만 label 태그로 렌더링
<Text as="label" size="small" weight="medium">이름</Text>
```
