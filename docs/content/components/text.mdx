# Text

일관된 타이포그래피를 적용하는 텍스트 렌더링 컴포넌트입니다.
`as` prop으로 시멘틱 HTML 요소를 지정할 수 있어 접근성과 SEO를 모두 고려한 마크업이 가능합니다.

## Props

### `as`

렌더링할 HTML 요소를 지정합니다. 시각적 스타일과 시멘틱 요소를 분리하여 사용할 수 있습니다.

| 값 | 기본값 | 설명 |
| --- | :---: | --- |
| `p` | ✓ | 일반 단락 텍스트 |
| `span` | | 인라인 텍스트 |
| `div` | | 블록 컨테이너 |
| `h1` ~ `h6` | | 제목 요소 |
| `label` | | 폼 레이블 |
| `li`, `ul`, `ol` | | 목록 요소 |

### `size`

텍스트의 크기를 결정합니다. 디자인 토큰에 정의된 타이포그래피 스케일을 따릅니다.

| 값 | 기본값 | 설명 |
| --- | :---: | --- |
| `display-large` | | 랜딩 페이지 히어로 영역의 대형 제목 |
| `display-medium` | | 페이지 단위 대형 제목 |
| `display-small` | | 섹션 단위 대형 제목 |
| `xlarge` | | 주요 섹션 제목 |
| `large` | | 서브 제목, 카드 제목 |
| `medium` | ✓ | 본문 텍스트 기본 크기 |
| `small` | | 보조 텍스트, 설명 |
| `xsmall` | | 캡션, 라벨, 힌트 |

### `weight`

폰트 두께를 결정합니다.

| 값 | 기본값 | CSS 값 | 설명 |
| --- | :---: | --- | --- |
| `regular` | ✓ | 400 | 본문, 설명 텍스트 |
| `medium` | | 500 | 강조가 필요한 본문 |
| `semibold` | | 600 | 서브 제목, 라벨 |
| `bold` | | 700 | 주요 제목, 강한 강조 |

### `intent`

텍스트의 색상 의도를 결정합니다.

| 값 | 기본값 | 설명 |
| --- | :---: | --- |
| `inherit` | ✓ | 부모 요소의 색상을 상속합니다. |
| `primary` | | 주요 콘텐츠 텍스트 색상 |
| `secondary` | | 보조 콘텐츠 텍스트 색상 |
| `success` | | 성공·완료를 나타내는 색상 |
| `warning` | | 경고를 나타내는 색상 |
| `danger` | | 오류·위험을 나타내는 색상 |
| `neutral` | | 중립적인 색상 |

### `align`

텍스트 정렬을 결정합니다.

| 값 | 기본값 | 설명 |
| --- | :---: | --- |
| `left` | ✓ | 왼쪽 정렬 |
| `center` | | 가운데 정렬 |
| `right` | | 오른쪽 정렬 |

### `truncate`

`true`로 설정하면 텍스트가 컨테이너를 넘칠 때 `...`으로 잘라냅니다.

- **타입**: `boolean`
- **기본값**: `false`

## 사용 가이드

### 시멘틱 마크업 유지하기

`size`가 시각적 크기를 제어하고 `as`가 HTML 요소를 결정합니다. 두 가지가 독립적으로 동작하므로 SEO·접근성을 위한 적절한 시멘틱 구조를 유지하면서 디자인을 자유롭게 적용할 수 있습니다.

```tsx
// 시각적으로 large이지만 h2 태그로 렌더링
<Text as="h2" size="large" weight="semibold">섹션 제목</Text>

// 시각적으로 small이지만 label 태그로 렌더링
<Text as="label" size="small" weight="medium">이름</Text>
```

## 코드

```tsx
import { Text } from "pine-design-system";

// 페이지 제목
<Text as="h1" size="display-small" weight="bold">Pine Design System</Text>

// 본문
<Text size="medium">컴포넌트 설명이 들어갑니다.</Text>

// 캡션
<Text as="span" size="xsmall" intent="neutral">마지막 수정: 2025.01.01</Text>

// 말줄임
<Text truncate style={{ maxWidth: 200 }}>길어지는 텍스트는 자동으로 줄임처리됩니다.</Text>
```
