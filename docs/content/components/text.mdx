# Text

토큰 기반 타이포그래피 컴포넌트입니다. `as`로 렌더링할 HTML 요소를 지정할 수 있습니다.

## 예시

<ComponentPreview>
  <TextDemo />
</ComponentPreview>

## Props

| Prop     | Type     | Default   | 설명 |
| -------- | -------- | --------- | ---- |
| as       | `"h1"` \| `"h2"` \| … \| `"p"` \| `"span"` \| `"div"` \| `"label"` 등 | `"p"` | 렌더링할 요소 |
| size     | Font size token (xsmall, small, medium, large, xlarge, display-*) | `"medium"` | 글자 크기 |
| weight   | `"regular"` \| `"medium"` \| `"semibold"` \| `"bold"` | `"regular"` | 굵기 |
| intent   | ColorIntent \| `"inherit"` | `"inherit"` | 텍스트 색상 의도 |
| align    | `"left"` \| `"center"` \| `"right"` | `"left"` | 정렬 |
| truncate | `boolean` | `false` | 말줄임 적용 |

## 코드

```tsx
import { Text } from "pine-design-system";

<Text as="h2" size="large" weight="bold">Heading</Text>
<Text intent="neutral">Body text</Text>
```
