# Switch

A toggle control for turning settings or features on and off. Built on [Base UI Switch](https://base-ui.com/react/components/switch) for accessibility.

Unlike Checkbox, Switch is for changes that take effect immediately. It fits cases where settings are applied without a separate save action.

## Guidelines

### Switch vs Checkbox

<DocTable
  headers={["Use case", "Recommended"]}
  rows={[
    ["On/off takes effect immediately (no save)", <strong>Switch</strong>],
    ["Reflected on form submit", <strong>Checkbox</strong>],
    ["Multiple selection from many options", <strong>Checkbox</strong>],
    ["Single setting toggle", <strong>Switch</strong>],
  ]}
  align={["left", "left"]}
/>

### Accessibility

Always provide a label via the `label` prop so screen readers can identify the purpose of the switch.
