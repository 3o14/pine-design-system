# Foundation

Pine Design System의 시각적 스타일은 **시맨틱 토큰**으로 정의됩니다. 각 테마(basic, game, crayon)가 이 토큰에 값을 채우기 때문에, 컴포넌트 코드를 바꾸지 않고도 디자인만 전환할 수 있습니다.

## 토큰 구조

- **Color**: 의도별 색상(primary, secondary, success, warning, danger, neutral)과 surface 색상
- **Typography**: 폰트 패밀리, 크기, 줄높이, 굵기
- **Spacing**: 간격 스케일 (xxxs ~ xxxl)
- **Radius**: 모서리 반경 (none, small, medium, large, xlarge, full)
- **Shadow**: 그림자 (none, xsmall ~ xlarge, pixelBox 등)
