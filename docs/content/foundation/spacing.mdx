# Spacing

간격(spacing) 토큰은 패딩, 마진, 갭 등 레이아웃과 컴포넌트 내부 여백을 일관되게 정의합니다.

## 개요

Pine에서는 `none`부터 `xxxl`까지 10단계의 spacing 토큰을 제공합니다. 픽셀 값은 고정되어 있으며, 컴포넌트 스타일과 레이아웃에서 이 토큰을 사용하면 테마 전환 시에도 일관된 비율을 유지할 수 있습니다.

## Spacing 토큰

<SpacingScale />

Preview 막대 길이는 토큰 값(px)에 비례합니다.

## 사용 가이드라인

- **토큰 우선**: 숫자(예: 16) 대신 토큰(md)을 사용해 스케일을 유지합니다.
- **컴포넌트**: Button, Card, TextField 등의 padding/radius는 테마에서 위 foundation spacing으로 매핑됩니다.
