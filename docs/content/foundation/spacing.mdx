# Spacing

간격은 **시맨틱 스케일**로 제공됩니다. 픽셀 값은 테마에서 정의되며, 기본 스케일은 다음과 같습니다.

## 스케일

| 토큰 | 기본값 (참고) |
| ---- | ------------- |
| none | 0px           |
| xxxs | 2px           |
| xxs  | 4px           |
| xs   | 8px           |
| sm   | 12px          |
| md   | 16px          |
| lg   | 24px          |
| xl   | 32px          |
| xxl  | 40px          |
| xxxl | 64px          |

컴포넌트 스타일과 레이아웃에서 이 토큰을 사용하면 테마 전환 시에도 일관된 비율을 유지할 수 있습니다.
