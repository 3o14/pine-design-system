# Colors

색상은 **의도(intent)** 단위로 정의되며, 각 의도마다 surface / text / border / weak 등 변형이 있습니다.

## Color intents

| Intent     | 용도 예시                    |
| ---------- | ---------------------------- |
| primary    | 메인 CTA, 강조               |
| secondary  | 보조 액션, 덜 강조된 요소     |
| success    | 성공, 완료, 확인             |
| warning    | 경고, 주의                   |
| danger     | 삭제, 오류, 위험             |
| neutral    | 배경, 테두리, 비강조 텍스트   |

각 intent에는 다음 토큰이 있습니다.

- **surface** — 배경
- **surfaceHover** — 호버 시 배경
- **surfaceActive** — 활성/누름 시 배경
- **text** — 텍스트
- **border** — 테두리
- **weak** — 약한 강도(배경/텍스트)

## Surface colors

화면 전체 배경·텍스트용 토큰입니다.

- **background** — 기본 배경
- **backgroundElevated** — 카드·모달 등 상위 레이어 배경
- **outline** — 아웃라인/포커스 링
- **text** — 기본 텍스트
- **textMuted** — 보조 텍스트
- **divider** — 구분선

컴포넌트의 `intent` prop(예: Button의 `intent="primary"`)은 이 토큰과 연결됩니다.
