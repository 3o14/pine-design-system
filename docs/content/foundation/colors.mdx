# Colors

색상은 **Foundation 팔레트**를 기반으로 시맨틱 **intent** 토큰에 매핑됩니다.
각 테마(basic, game, crayon)는 동일한 foundation 팔레트를 공유하되,
game 테마는 `toNeon()`, crayon 테마는 `toPastel()` 어댑터로 파생됩니다.

## 테마 어댑터

| 테마   | 어댑터     | 동작                                      |
| ------ | ---------- | ----------------------------------------- |
| basic  | —          | Foundation 팔레트 직접 참조               |
| game   | `toNeon()` | Saturation +30%, Lightness 상향 (네온 효과) |
| crayon | `toPastel()` | Saturation ×0.65, Lightness 상향 55% (파스텔 효과) |

## Intent 가이드

| Intent     | 용도                           |
| ---------- | ------------------------------ |
| primary    | 메인 CTA, 강조 요소            |
| secondary  | 보조 액션, 덜 강조된 요소      |
| success    | 성공, 완료, 확인               |
| warning    | 경고, 주의                     |
| danger     | 삭제, 오류, 위험               |
| neutral    | 배경, 테두리, 비강조 텍스트    |

각 intent에는 다음 state 토큰이 있습니다.

- **surface** — 기본 배경색
- **surfaceHover** — 호버 시 배경색
- **surfaceActive** — 클릭/활성 시 배경색
- **text** — 해당 배경 위에 올라오는 텍스트 색
- **border** — 테두리 색
- **weak** — 약한 강도의 배경 (subtle 배지 등)

컴포넌트의 `intent` prop은 이 토큰과 직접 연결됩니다.
