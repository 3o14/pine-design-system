# Introduction

Pine Design System은 **토큰 기반 테마**와 **접근성**을 갖춘 React UI 컴포넌트 라이브러리입니다.

## 한 번에 바꾸는 디자인 컨셉

`ThemeProvider`의 **`design`** prop 하나로 앱 전체의 디자인 컨셉을 바꿀 수 있습니다. 코드는 그대로 두고, basic / game / crayon 중 하나를 선택하면 됩니다.

## Base UI 기반

컴포넌트는 [Base UI](https://base-ui.com/)의 headless 프리미티브를 기반으로 합니다. 마크업과 키보드/스크린 리더 동작은 Base UI가 담당하고, Pine은 토큰과 스타일만 적용합니다. 따라서 접근성과 동작 일관성을 유지하면서도 디자인만 자유롭게 바꿀 수 있습니다.

## 다크/라이트 모드

`theme` prop으로 **light** / **dark** 모드를 지정할 수 있고, `syncWithSystem={true}`(기본값)로 OS 설정을 따르게 할 수 있습니다. 모든 테마(basic, game, crayon)가 라이트/다크 모드를 지원합니다.
